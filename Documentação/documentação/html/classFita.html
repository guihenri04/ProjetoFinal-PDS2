Detailed Description

Na classe "Fita", que herda publicamente a classe "Filme", existe como atributo privado a variável booleana "rebobinada", que determina se a fita foi rebobinada ou não. A classe também contém como atributos públicos um construtor, o qual recebe como parâmetros um id de identificação de uma fita, seu título e a quantidade de unidades dela disponíveis para locação ("Fita(int id, string titulo, int unidades)"), um método para determinar se a fita foi rebobinada ou não ("void rebobinar()"), um método para calcular quanto será pago pelo aluguel da fita de acordo com a quantidade de dias que ela ficou alugada ("("int calcularValor(int dias) override")") e um método para alugar as fitas disponíveis na locadora ("void serAlugado() override"). Ambos métodos substituem funções virtuais na classe base, no caso,"Filme".

int Fita::calcularValor ( int  dias)
overridevirtual

Método "calcularValor" da classe "Fita", utilizado especialmente para controle e registro.

Esse método calcula quanto o cliente terá de pagar pelo aluguel da fita de acordo com a quantidade de dias que ela ficou alugada. Caso o cliente devolva a fita já rebobinada, o aluguel é fixo em 5 reais, caso contrário, é fixo em 7 reais.

Parameters
diasArmazena quantos dias um determinado filme (no caso, fita) ficou alugado.
Returns
Retorna o preço que o cliente terá de pagar pelo aluguel da fita, sendo 5 reais fixos o preço da locação de uma fita rebobinada e 7 reais fixos o de uma não rebobinada.
