\doxysubsection{Detailed Description}
Na classe \char`\"{}\+Locadora\char`\"{},existem como atributos públicos um construtor (\char`\"{}\+Locadora()\char`\"{}) e um destrutor (\char`\"{}$\sim$\+Locadora()\char`\"{}). Existe também um método para cadastrar filmes, que recebe como parâmetros o tipo do filme, a quantidade dele, seu id de identificação, seu título e sua e sua categoria, seja estoque, promoção ou lançamento (\char`\"{}void cadastrar\+Filme(char tipo, int quantidade, int id, const string\& titulo, const char\& categoria)\char`\"{}), um método para remover filmes de acordo com seu código (\char`\"{}void remover\+Filme(int codigo)\char`\"{}), um método para listar filmes de de acordo com a opção selecionada, seja a partir do id de identificação ou título do filme (\char`\"{}void listar\+Filmes(char opcao)\char`\"{}), um método para cadastrar clientes, de acordo com o cpf e o nome informados (\char`\"{}void cadastrar\+Cliente(long long cpf, string nome)\char`\"{}), um método para remover clientes utilizando somente o cpf informado (\char`\"{}void remover\+Cliente(long long cpf)\char`\"{}), um método para listar clientes a partir da opção selecionada, seja por nome ou cpf cadastrados (\char`\"{}void listar\+Clientes(char opcao)\char`\"{}), um método para exibir uma lista de sobre clientes bloqueados na locadora (\char`\"{}void listar\+Bloqueados()\char`\"{}), um método para registrar os clientes que alugaram filmes e quais filmes foram alugados, com base em seus id\textquotesingle{}s (\char`\"{}void aluguel(long long cpf, vector $<$int$>$ id)\char`\"{}), um método para controlar quais clientes fazem a devolução dos filmes e com quantos dias esse cliente ficou com o filme alugado (\char`\"{}void devolucao(long long cpf, int dias)\char`\"{}), um método para recomendar filmes aos clientes, de acordo com o cpf informado (\char`\"{}void recomendar\+Filmes(long long cpf)\char`\"{}) e um método para avaliar os filmes, de acordo com seu id e a nota dada (\char`\"{}void avaliar\+Filme(int id, int nota)\char`\"{}). Além disso, a classe conta um vetor de ponteiros para armazenar os filmes da locadora (\char`\"{}vector$<$\+Filme$\ast$$>$ filmes\char`\"{}) e um vetor de ponteiros para armazenar os clientes da locadora (\char`\"{}vector$<$\+Cliente$\ast$$>$ clientes\char`\"{}). Por fim, existe um vetor de ponteiros que armazena uma lista de clientes ordenados (\char`\"{}vector$<$\+Cliente$\ast$$>$ clientes\+Ordenados\char`\"{}) e outro vetor, também de ponteiros, que armazena uma lista de filmes ordenados (\char`\"{}vector$<$\+Filme$\ast$$>$ filmes\+Ordenados\char`\"{}).

Método \char`\"{}recomendar\+Filmes\char`\"{} da classe \char`\"{}\+Locadora\char`\"{}. 

O método recebe como parâmetro um cpf e procura no vetor \char`\"{}clientes\char`\"{} pelo cliente com o cpf informado. Se o cliente não for encontrado, uma mensagem de erro é exibida. Caso contrário, ocorre a chamada do método \char`\"{}recomendar()\char`\"{} para o cliente em questão, o qual recomenda filmes com base nos gostos desse cliente e nos gostos de outros clientes da locadora que são considerados similares à ele. Esse método remove um filme da locadora. Ele recebe como parâmetro o id de identificação do filme e, a partir daí, percorre o vetor \char`\"{}filmes\char`\"{} procurando o filme com esse id, utilizando a funcão \char`\"{}find\+\_\+if\char`\"{}. Caso esse filme seja encontrado, o método o retira do vetor \char`\"{}filmes\char`\"{} e, portanto, da locadora. Caso contrário, o método exibe uma mensagem de erro e não retira nenhum filme da locadora.
