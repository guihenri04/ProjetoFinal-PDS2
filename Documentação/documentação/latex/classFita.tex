\doxysubsection{Detailed Description}
Na classe \char`\"{}\+Fita\char`\"{}, que herda publicamente a classe \char`\"{}\+Filme\char`\"{}, existe como atributo privado a variável booleana \char`\"{}rebobinada\char`\"{}, que determina se a fita foi rebobinada ou não. A classe também contém como atributos públicos um construtor, o qual recebe como parâmetros um id de identificação de uma fita, seu título e a quantidade de unidades dela disponíveis para locação (\char`\"{}\+Fita(int id, string titulo, int unidades)\char`\"{}), um método para determinar se a fita foi rebobinada ou não (\char`\"{}void rebobinar()\char`\"{}), um método para calcular quanto será pago pelo aluguel da fita de acordo com a quantidade de dias que ela ficou alugada (\char`\"{}(\char`\"{}int \mbox{\hyperlink{classFita_a78674c33107135606cca34e140b31e84}{calcular\+Valor(int dias)}} override\char`\"{})\char`\"{}) e um método para alugar as fitas disponíveis na locadora (\char`\"{}void ser\+Alugado() override\char`\"{}).
