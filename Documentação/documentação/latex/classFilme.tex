\doxysubsection{Detailed Description}
Na classe \char`\"{}\+Filme\char`\"{}, a qual possui como classe amiga \char`\"{}\+Locadora\char`\"{} e \char`\"{}\+Cliente\char`\"{}, existem como atributos protegidos quantas vezes um determinado filme foi avaliado (\char`\"{}int vezes\+Avaliado\char`\"{}), o nome do filme (\char`\"{}string titulo\char`\"{}) e o tipo desse filme, seja fita, D\+VD, etc (\char`\"{}string tipo\char`\"{}). A classe também contém como atributos públicos um construtor que recebe o id, o nome e a quantidade disponível de determinado filme (\char`\"{}\+Filme(int id, string titulo, int unidades)\char`\"{}) e um destrutor virtual, inicializado como default (\char`\"{}$\sim$\+Filme()\char`\"{}). Existe também um método para calcular quanto será pago pelo aluguel do filme, de acordo com o tempo de aluguel (\char`\"{}virtual int calcular\+Valor(int dias) = 0\char`\"{}), um método para identificar os filmes (\char`\"{}void ler\+Filme()\char`\"{}), um método para alugar os filmes disponíveis (\char`\"{}virtual void ser\+Alugado()\char`\"{}), um método para devolver os filmes que foram alugados (\char`\"{}void ser\+Devolvido()\char`\"{}) e um método para avaliar o filme alugado (\char`\"{}void ser\+Avaliado(int nota)\char`\"{}). Além disso, contém um atributo público para controlar a quantidade de filmes disponíveis na locadora (\char`\"{}int unidades\char`\"{}), outro atributo para registrar a nota de avaliação recebida pelo filme (\char`\"{}float avaliação\char`\"{}) e, por fim, um atributo que para registrar o id de identificação de um filme (\char`\"{}int id\char`\"{}).
